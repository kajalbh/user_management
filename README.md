# User Management System

Mini Project Management REST APIs built using Spring Boot.

## Tech Stack
- Java 17
- Spring Boot 3
- Spring Security (JWT)
- Spring Data JPA
- MySQL
- Maven

## Features
- User registration and login
- JWT-based authentication and authorization
- Create, update, view, and delete projects
- Manage tasks under projects
- Task search and sorting
- Input validation
- Global exception handling

## Modules
- Auth Controller
- Project Controller
- Task Controller

## How to Run
1. Clone the repository
2. Open project in IntelliJ IDEA
3. Configure database in 'application.properties'
